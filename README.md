# 🚦 Tesi Universitaria

## Controllo semaforico adattivo mediante strategie Max-Pressure: progettazione e valutazione tramite simulazione

### Argomento

La tesi esplora il **controllo adattivo del traffico urbano** utilizzando l'algoritmo **Max-Pressure**, una strategia decentralizzata che minimizza la congestione regolando dinamicamente i tempi semaforici in base alla pressione misurata sulle corsie.

### Metodologia

- **Progettazione** del controllore Max-Pressure per intersezioni semaforizzate
- **Implementazione** in ambiente di simulazione (SUMO / Aimsun / VISSIM)
- **Valutazione** comparativa delle performance: throughput, ritardi, code, emissioni
- **Confronto** con strategie tradizionali (fixed-time, actuated)

### Strumenti previsti

- **SUMO** (Simulation of Urban MObility) — simulatore open-source
- **Python** — scripting e analisi dati
- **TraCI** — interfaccia Python-SUMO
- **Matplotlib / Pandas** — visualizzazione risultati
